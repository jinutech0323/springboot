HTTP Request : Message from client to server

Http request -> Start line -> URL 
					    -> HTTP Methods -> GET,POST,PUT,DELETE 
					    -> HTTP Version -> 
                       ->Headers -> additional information to the server 
		       ->Body  -> will send data to server 

GET -> Fetching a data from the server (Reading)
POST -> Creating the data in the Server (Create)
PUT -> Updating the data (Update) 
DELETE -> Delete the data from server (Delete)

HTTP Response : Message from server to client 

Http response -> Status line -> HTTP Version 
						 -> Status Code 
						 -> Status text
			 -> Headers -> information
			 ->Body -> data

Status code

1. 1XX -> 100 -> 199  -> Informational 
2. 2XX -> Success 
3. 3XX -> Redirectional (305)
4. 4XX -> Client error (404,405)
5. 5XX -> Server error
